---

## 📌 Project Overview

The Smart Student Management System is a web application that helps manage student records, attendance, marks, authentication, and dashboard operations.

It provides separate functionalities for administrators and students with a clean and responsive user interface.

---

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 👨‍🎓 Student Management
- ➕ Add Student
- ✏️ Update Student
- ❌ Delete Student
- 🔍 Search Student
- 📊 Dashboard
- 📄 Student List
- 🌐 REST APIs
- 📱 Responsive UI

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yuvimandloi/Smart-Student-Management-System.git
```

### Backend

```
Open Backend in IntelliJ IDEA

Run Spring Boot Application
```

### Frontend

```
cd frontend

npm install

npm start
```
